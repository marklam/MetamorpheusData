Running some example data through MetaMorpheus with different settings and databases.

The data files are `TEMP_1_DIA_45.raw`, `TEMP_2_DIA_46.raw` and `TEMP_3_DIA_47.raw` of the dataset at [https://www.ebi.ac.uk/pride/archive/projects/PXD020127](https://www.ebi.ac.uk/pride/archive/projects/PXD020127) (from the paper [https://www.sciencedirect.com/science/article/pii/S2095177921000502](https://www.sciencedirect.com/science/article/pii/S2095177921000502))

A search with the default parameters and no calibration, with and without protein grouping is run with the output in sub-folders of Results.
